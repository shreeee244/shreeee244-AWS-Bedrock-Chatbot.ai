# 🤖 AWS Bedrock Chatbot

A terminal-based AI chatbot using AWS Bedrock and Amazon Nova Lite.

## 🚀 Quick Start
-----------------------------------------------------------------------------------------------------------------------------------------------------
💬 Commands
Type your message to chat

clear - Reset conversation

quit - Exit

💰 Cost
~₹0.025 per 1,000 messages (practically free)

📁 Files
chatbot.py - Main chatbot

test_bedrock.py - Test script

requirements.txt - Dependencies
-----------------------------------------------------------------------------------------------------------------------------------------------------
🛠️ Built With
AWS Bedrock

Amazon Nova Lite

Python Boto3
